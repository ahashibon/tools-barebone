About section is compulsory. To customize the text here, please

  • create a template file with below HTML content 
    <div id="accordion">
    <h2>What does this tool do?</h2>
    <div>
        Add information about this tool here...
    </div>
</div>
  • store it in the folder for custom templates in the path templates/user_templates
  • in the config file, create a templates dictionary, if not yet there, and set the variable about to point to you custom template: 
    templates:
   - about: "custom_about.html"

If you want to add more section(s) after about, please follow the above steps again for every new section.
